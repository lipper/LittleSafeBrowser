# LittleSafeBrowser
A safe browser for little child , config on age , write mainly in Python , Qt , have fun!
First of all, I am a Chinese man, so the following words should be wrote by Chinese first.

# 功能设计：
利用pyQt做主界面的开发
包括以下功能模块：

## 1、网址簿：保存在本地ES中，方便管理，另外联网时可以与云上服务同步更新，lb浏览器中以列表方式访问，不设地址输入栏，全部预先按年龄/年龄段定制。
## 2、网页内容（文本）过滤，Get到本地的页面内容首先进行分词，并对其中包含的不健康内容进行滤除。关键词列表从ES中取出放入到redis中缓存，提高过滤效率。
## 3、用户在点击内容中的链接时，如果该url不在许可列表内，会禁止访问，并弹出对话框请求家长授权（可考虑通过手机客户端给予授权，如微信等app中）。
## 4、家长管理：许可网址列表管理、停用词列表管理、浏览器的限时访问时间段设置，配置可以按用户上传到服务器（前期不加限制）。
ps：不集成在lb界面中，防止孩子过于聪明，突破限制后自己更改。
## 5、挑战功能：在许可时间之外，lb显示锁定状态，但是允许家长设置选项，是否容许孩子通过解谜游戏（如数独）来解锁，如果容许，当孩子通关后，当前时间段限制解除！游戏环节可以增进兴趣是不是呢？
## 6、定时提醒眼保健操功能，在使用时间超过1小时后，会自动提示放松眼睛，屏幕会自动调成保护色，立体拉远、拉近效果，调节眼部肌肉，达到放松效果，如果选择眼保健操，则播放眼保健操视频/音频，记录孩子选择的项，下次同一时段会按统计结果指定默认选项，倒数3秒自动进入。
## 7、使用时间段统计功能，类似时间轴或甘特图，供家长调整设置参考。
